# Low Visibility — Brand Bible

A digital brand system and editorial archive for **Low Visibility**, developed through **Simple Minds Studios** by Bart Salazar.

## About the Project

This repository presents the visual language, materials, palette, typography, campaign logic, and creative philosophy behind the Low Visibility identity. It is designed as a living brand reference rather than a generic template or downloadable design kit.
